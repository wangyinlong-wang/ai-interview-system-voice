## 1. 测试概述

### 1.1 测试范围

本次补充测试覆盖 AI 模拟面试系统 V3 版本新增的定时爬虫与面试知识库模块，具体包括：

| 模块类别 | 具体模块 | 测试重点 |
|----------|----------|----------|
| 数据库模型 | QuestionBank、CrawlTask、CrawlSource 表 | 字段完整性、索引正确性、与现有表兼容性 |
| 爬虫模块 | 爬虫基类、牛客网爬虫、力扣爬虫、GitHub爬虫 | 请求重试、页面解析、反爬策略、异常处理 |
| 数据管道 | 清洗、去重(SimHash)、分类、存储 | 数据质量、去重精度、分类准确率、入库一致性 |
| 定时调度器 | APScheduler 调度器 | 初始化、定时触发、手动触发、动态任务管理 |
| API接口 | 题库接口(/questions)、采集管理接口(/admin/crawl) | 参数校验、权限控制、响应格式、分页搜索 |
| AI集成 | 知识库检索、题目引用、上下文注入 | 检索相关性、引用计数、降级处理 |
| 前端页面 | 题库浏览页、管理后台、采集管理页 | 页面加载、交互功能、数据展示、操作反馈 |
| 集成测试 | 端到端流程验证 | 调度→采集→清洗→入库→展示→审核 全链路 |
| 性能测试 | 爬虫速度、去重效率、API响应时间 | 并发性能、大数据量处理、3D场景兼容性 |
| 安全合规 | 爬虫合规、注入防护、权限控制 | robots.txt遵守、请求频率、数据安全 |

| **XSS存储型攻击** | 在题目标题中输入`<script>alert('xss')</script>` | React自动转义，不执行脚本 | 前端转义有效 | 低风险 |
| **越权访问管理接口** | 未登录访问/admin/crawl/* | 返回401未授权 | JWT校验拦截 | 低风险 |
| **爬虫请求频率控制** | 检查代码中delay配置和实际请求间隔 | 间隔>=1秒，符合要求 | 1-3秒随机延迟 | 低风险 |
| **错误信息脱敏** | 触发各类异常，检查API返回 | 不包含堆栈/路径/SQL等敏感信息 | 错误信息脱敏 | 低风险 |
| **robots.txt合规** | 检查User-Agent和预留接口 | UA可识别，代码结构支持扩展 | 基础合规，需生产环境完善 | 中风险 |
