# AI 模拟面试系统 — 项目计划

## 项目概述
开发一套 AI 模拟面试系统，核心功能：
- 用户上传简历（PDF/DOC/文本）
- 系统解析简历内容
- 结合常见面试问题 + 简历个性化问题
- AI 扮演面试官进行实时模拟面试
- 面试后给出评分和改进建议

## 团队角色与交付物

### Stage 1 — 需求分析（产品经理）
- **输出**: PRD文档（需求规格说明书）
- **内容**: 用户画像、功能模块、用户故事、流程图、非功能需求

### Stage 2 — 技术设计（架构师）
- **输出**: 技术架构文档
- **内容**: 系统架构图、技术栈选型、数据库设计、API 设计、AI 集成方案

### Stage 3 — 项目开发（前后端开发）
- **输出**: 可运行的全栈应用
- **内容**: 
  - 前端：React + TypeScript 面试界面
  - 后端：API 服务、简历解析、AI 对话

### Stage 4 — 测试验收（测试工程师）
- **输出**: 测试报告
- **内容**: 测试用例、自动化测试、Bug 报告、验收清单

## 技术栈预选型（架构师确认）
- 前端: React 19 + TypeScript + Vite + Tailwind CSS + shadcn/ui
- 后端: Python FastAPI
- AI: 大模型 API（模拟面试对话 + 简历解析）
- 存储: 本地 JSON / SQLite
- 部署: 静态网站 + Python 服务
